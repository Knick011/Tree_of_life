# Tree of Life (TOL)

Guided prescribing speed tool for primary care clinics. Patient-fit checks, EMR-ready copy output, doctor templates.

## Live demo

Once GitHub Pages is enabled: https://knick011.github.io/Tree_of_life/

## Run locally

```bash
node server.js
```

Then open http://127.0.0.1:4173

No build step. Vanilla JS, static files only.

## Status

Beta — clinical content is mock/draft. Not for clinical use.

See `DATA-ARCHITECTURE.md` for the data model and Canadian data source plan.
